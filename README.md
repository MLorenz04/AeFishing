# AeFishing

Jednoduchý český plugin pro přidání vylepšeného rybaření do minecraftu

Přidává možnost:

* Chytit 21 nových ryb, které mají různé délky
* Poklady pěti různých kategorií
  * Obyčejné
  * Neobyčejné
  * Vzácné
  * Epické
  * Legendární

Seznam ryb:
* Amur bílý
* Bolen dravý
* Candát obecný
* Cejn siný
* Drsek menší
* Hlavatka mramorovaná
* Hrouzek běloploutvý
* Jelec jesen
* Jeseter malý
* Ježdík žlutý
* Kapr obecný
* Mník jednovousý
* Lipan podhorní
* Parma obecná
* Perlín ostrobřichý
* Pstruh obecný
* Sekavčík horský
* Sumec velký
* Štika obecná
* Vranka obecná
* Vyza velká
